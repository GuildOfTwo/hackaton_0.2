# Onboardy
Onboardy - это сервис, который автоматизирует и упрощает процесс онбординга и адаптации новых сотрудников. Проект выполнен в рамках хаккатона [Лидеры цифровой трансформации Краснодарский край](https://i.moscow/lct/krasnodar). MVP сервиса доступен по ссылке.
